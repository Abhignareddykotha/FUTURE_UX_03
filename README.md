# 🚀 CRM Dashboard UI Design — PixelCraft Web Studio

## 📌 Project Description
This project is a complete **CRM Dashboard UI design** created for a digital agency named **PixelCraft Web Studio**.

The dashboard helps teams manage leads, track projects, monitor sales performance, handle client information, and manage follow-ups efficiently.

The goal was to design a **modern, workflow-driven and user-friendly interface** that improves productivity and decision-making for agency teams.

---

## 🎯 Problem Statement
Many agencies struggle with:

- Managing leads across multiple stages  
- Tracking client projects and revenue  
- Missing follow-ups and deadlines  
- Lack of performance insights  
- Switching between multiple tools  

This CRM dashboard provides a **centralized solution**.

---

## 🎯 Project Objective
The main objective is to design a **clean and efficient CRM interface** that:

- Helps agencies visually manage sales pipeline  
- Provides quick analytics insights  
- Supports client relationship management  
- Enables task and follow-up tracking  
- Improves workflow efficiency  

---

## 👥 Target Users

- Digital marketing agencies  
- Web design & development studios  
- Freelancers managing multiple clients  
- Sales and operations teams  

---

## 🖥️ Screens Designed

### 🏠 Dashboard
- Overview cards (Total Leads, Active Clients, Running Projects, Tasks Due)  
- Leads by Stage graph  
- Sales Funnel visualization  
- Mini Pipeline board  

### 📋 Leads Page
- Lead table with status tags  
- Lead value and source tracking  
- Search and filter functionality  

### 🔄 Pipeline Board
- Kanban layout for deal tracking  
- Stages: New → Contacted → Qualified → Proposal → Negotiation → Won  

### 📊 Analytics Page
- Monthly Revenue bar chart  
- Pipeline performance radar chart  
- Lead stage distribution donut chart  

### 📁 Projects Page
- Project progress cards  
- Deadline and status indicators  

### 👤 Clients Page
- Client directory table  
- Active projects count  
- Revenue contribution  

### 📌 Client Profile View
- Contact information  
- Financial overview  
- Associated projects  

### ✅ Tasks & Follow-up Page
- Task list with priority labels  
- Filters: All, To Do, In Progress, Completed, Overdue  
- Due date tracking  

### ⚙️ Settings Page
- Profile settings  
- Notification toggles  
- Appearance options (Dark / Light)

---

## 🔄 Workflow Logic

1. Leads are added and tracked in the **Leads page**
2. Leads move across stages in the **Pipeline board**
3. Converted leads become **Clients**
4. Clients are assigned **Projects**
5. Teams manage **Tasks & follow-ups**
6. Performance is monitored through **Analytics**

---

## 🎨 UX Design Decisions

- Dark modern theme for reduced eye strain  
- Kanban pipeline for intuitive workflow  
- Color-coded status tags for clarity  
- Card-based layout for better hierarchy  
- Minimal sidebar navigation  
- Data visualization for quick decision making  

---

## 💡 How This Dashboard Improves Efficiency

- Centralized lead & client management  
- Reduced missed follow-ups  
- Faster decision making with analytics  
- Improved team productivity  
- Better workflow visibility
